This world is named as "DC" probably because there is a world named "CC" which serves as my construction lab. I think you can now consider it as the abbreviation of "Doghead County". This Doghead County, should be consisted of several cities, each have its unique building style and theme. The one I'm currently working on is a "Southen California city".

这个世界之所以被称为DC，可能是因为之前出现了一个代号为 CC的为我所用的建筑试验场。当然，你现在也可以认为DC是“doghead county”（狗头县）的简写。 DC将会由几个有着不同建筑风格的城市组成。当前正在建设中的是一个南加州风格的城市 （或者说小镇）。
